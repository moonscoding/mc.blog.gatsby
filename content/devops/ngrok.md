# ngrok



## command



다음 명령어로 ngrok을 설치해보세요.

```
brew cask install ngrok
```



설치가 완료됐는지 다음 명령어로 확인해보세요.

```
ngrok --help
```



ngrok 프록시를 시작해보세요.

```
ngrok http [port]
```

- ngrok http 8080
