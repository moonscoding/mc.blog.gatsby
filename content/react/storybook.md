## addon

Storybook에는 addon이라는 plugin 시스템이 존재하고 공식적으로 지원하는 addon 외에도 다양한 오픈소스 addon 이 존재합니다.



### addon_storyshots

snapshot test를 위한 addon.

```shell
$ yarn add --dev @storybook/addon-storyshots react-test-renderer
```



### addon_knobs

테스트 케이스를 동적으로 수정



### addon_viewpor

반응형을 쉽게 확인



### addon_notes

스토리북에 메모을 남길 수
